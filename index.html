Ugliest UI I’ve Seen

What’s wrong (brief)

  • Bad contrast and unreadable color choices
  • Inconsistent spacing, tiny clickable targets
  • Too many simultaneous visual elements (clutter)
  • Poor visual hierarchy — no clear primary action

How to improve it (short plan)

  1. Apply a consistent design system (colors, spacing, typography)
  2. Improve color contrast and accessibility (WCAG AA)
  3. Reduce clutter and define a clear visual hierarchy
  4. Use responsive breakpoints, bigger touch targets, and semantic HTML
